Pozoo Do Kinoingan

Klik butang di bawah untuk mengakses aplikasi Buku Lagu Pozoo melalui penyemak imbas web.

Buka Aplikasi (Localhost) Buka Aplikasi (LAN)

Cara Penggunaan:

  1. Pastikan Expo Metro Bundler telah dijalankan dengan perintah npm run dev:fix
  2. Klik butang "Buka Aplikasi" untuk mengakses aplikasi melalui web
  3. Jika anda mengakses dari peranti lain, gunakan butang "Buka Aplikasi (LAN)"

Alamat IP LAN anda: 192.168.0.143

Port: 8082
