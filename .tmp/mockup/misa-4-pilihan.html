Pilihan 1 — Outlined List

Konsep: Flat list dengan part label kat tepi kiri macam column table. Tiada card besar — setiap bahagian satu row panjang dengan horizontal divider. Sangat ringkas dan formal.

Fungsi

  • Tambah: "+ Tambah lagu" dashed line kat bawah setiap bahagian → modal search
  • Padam: ikon X merah kecik kat hujung setiap lagu
  • Kod lagu: kotak input 38px lebar inline, tekan untuk tulis (C, D, G, F, Em, Am...)
  • Susun: nombor kecil 1,2,3,4 kat kiri lagu (susunan visual, drag handle boleh ditambah)
  • Kosong: terus render "+ Tambah lagu" — tiada card besar
  • Header: Share & Clear icon kat kanan
✓ Multi-song OK: Pembukaan tunjuk 4 lagu dengan kod C / F / G / kosong.

Pilihan 2 — Chip Rows

Konsep: Setiap lagu jadi pill/chip kecil dengan semua info (ID, title, key, trash) inline. Lagu banyak pun muat sebab wrap ke baris baru automatik. Sangat compact & playful.

Fungsi

  • Tambah: chip "+ Tambah" ungu di hujung setiap row
  • Padam: X merah kecik dalam chip lagu
  • Kod lagu: kotak kecik dengan border kat dalam chip
  • Susun: chips kekal mengikut order dalam flex-wrap (long-press untuk swap boleh)
  • Kosong: satu chip "+ Tambah lagu" — tiada card tambahan
✓ Multi-song OK: 4 lagu dalam Pembukaan, wrap automatik ke 2 baris.

Pilihan 3 — Stepper Timeline

Konsep: Vertical timeline dengan dot bernombor kat tepi kiri, line penyambung antara step. Macam progress tracker. Body box kecil untuk lagu.

Fungsi

  • Tambah: "+ Tambah lagu" kecik kat bawah body box
  • Padam: X kecik kat hujung setiap lagu
  • Kod lagu: kotak 28px dengan border, inline kat kanan
  • Susun: songs kekal dalam order timeline (drag untuk swap boleh)
  • Kosong: dot jadi dashed, body box tunjuk "+ Tambah lagu" je
  • Visual progress: nampak flow 1→6 macam checklist
✓ Multi-song OK: Pembukaan ada 4 lagu, body expand ikut ketinggian.

Pilihan 4 — Card Grid 2-Col

Konsep: 6 bahagian Misa disusun dalam grid 2 lajur (3 baris × 2). Setiap bahagian satu tile kompak. Nampak macam "menu grid" — banyak info dalam satu skrin.

Fungsi

  • Tambah: "+ Tambah" box dashed dalam tile, atau "+ tambah lagi" pill bawah
  • Padam: long-press pada lagu (icon boleh ditambah)
  • Kod lagu: key chip kecik kat hujung setiap mini-song
  • Susun: tap tile → masuk full view untuk reorder
  • Kosong: tile dotted top, "+ Tambah" besar kat tengah
✓ Multi-song OK: Pembukaan tile ada 4 lagu, "+ tambah lagi" pill tunjuk masih boleh tambah.
