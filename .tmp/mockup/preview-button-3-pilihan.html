Kelebihan: Pantas, tak tinggal context, senang dismiss (tap luar).
Kelemahan: Limited info — tiada key penuh, tiada navigate.

B · Print Preview ScreenFull-screen putih macam nak share — tapi belum share.
Konsep: Tap ikon mata → skrin putih penuh macam dokumen printable/WhatsApp share. User nampak exactly macam mana hasilnya nanti kalau dia share.

Kelebihan: Preview exactly macam share-image, senang baca besar.
Kelemahan: Tinggal context skrin Misa (tapi boleh close).
